# FarmaLocal

> Sistema de gestão para farmácias locais — MVP em desenvolvimento.

## 📋 Sobre o Projeto

O **FarmaLocal** é uma aplicação web desenvolvida em **.NET** com o objetivo de facilitar a gestão de farmácias locais. O sistema contempla o controle de estoque de medicamentos, cadastro de clientes, registro de vendas e emissão de relatórios, proporcionando mais agilidade e organização no dia a dia da farmácia.

Este repositório contém o MVP (Produto Mínimo Viável) do projeto, com as funcionalidades essenciais para operação básica de uma farmácia de pequeno porte.

## 🚀 Funcionalidades Previstas (MVP)

- 📦 **Gestão de Estoque** — Cadastro, atualização e controle de medicamentos
- 👥 **Cadastro de Clientes** — Registro e consulta de clientes
- 🛒 **Registro de Vendas** — Controle de vendas e histórico de transações
- 📊 **Relatórios** — Geração de relatórios de estoque e vendas
- 🔐 **Autenticação** — Controle de acesso por perfil de usuário

## 🛠️ Tecnologias

- [.NET](https://dotnet.microsoft.com/) — Plataforma de desenvolvimento
- [ASP.NET Core](https://docs.microsoft.com/aspnet/core) — Framework web
- [Entity Framework Core](https://docs.microsoft.com/ef/core) — ORM para acesso a dados

## ▶️ Como Executar

### Pré-requisitos

- [.NET SDK](https://dotnet.microsoft.com/download) 8.0 ou superior

### Passos

```bash
# Clone o repositório
git clone https://github.com/rubenslyra/farmalocal.git

# Acesse o diretório do projeto
cd farmalocal

# Restaure as dependências
dotnet restore

# Execute a aplicação
dotnet run
```
